# TopCare AI Configuration System

Semua konfigurasi platform dikelola melalui Configuration Manager.

Contoh konfigurasi:

- Application
- Theme
- Language
- Feature Flag
- AI Provider
- Advertisement
- Analytics
- API Endpoint
- Membership
- Payment
- Storage
- Environment

Prinsip:

- Konfigurasi tidak boleh hardcoded di module.
- Semua module membaca konfigurasi dari Configuration Manager.
