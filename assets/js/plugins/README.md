# TopCare AI Plugins

Semua plugin pihak ketiga ditempatkan di folder ini.

Contoh:

- Google Analytics
- Google Ads
- Meta Pixel
- OpenAI
- Gemini
- Midtrans
- Stripe
- Facebook SDK
- YouTube API
- WhatsApp Business
- Enterprise Connector

Plugin tidak boleh mengubah Core.

Plugin hanya berkomunikasi melalui:

- Kernel
- Container
- Lifecycle
- Registry
